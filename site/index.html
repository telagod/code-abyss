How It Works

🎭

Identity

Who the agent is. Role anchoring, personality traits, emotional patterns. Each persona is a distinct character.

⚙️

Shared Behavior

How the agent works. Iron laws, execution chains, proactive protocol. Shared across all personas.

✍️

Output Style

How the agent speaks. Report skeletons, tone calibration. Template variables make any persona × style safe.

5 personas × 5 styles = 25 validated combinations, zero conflicts.

Install

Claude Code

npx code-abyss -t claude -y or claude plugin install code-abyss

Codex CLI

npx code-abyss -t codex -y

Gemini CLI

npx code-abyss -t gemini -y

OpenClaw

npx code-abyss -t openclaw -y

Mix persona and style: npx code-abyss -t claude --persona elder-sister --style abyss-cultivator -y
