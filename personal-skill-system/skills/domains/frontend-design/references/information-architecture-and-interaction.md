# Information Architecture And Interaction / 信息架构与交互

## 1. Structure Before Styling

Before colors or shadows, define:

- primary user goal
- primary screen action
- information hierarchy
- navigation depth

## 2. Interaction Rules

- one primary action per region
- avoid ambiguous motion
- preserve predictable control placement
- reduce mode switches

## 3. Review Questions

- can a user tell what matters first
- is there more than one competing primary action
- what interaction is likely to confuse first-time users
