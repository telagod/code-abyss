# Expert Motion And State Transitions

## Use

Use this when motion should reduce ambiguity instead of merely making the UI look alive.

## Rules

- motion explains change of state, ownership, or focus
- duration should scale with distance and complexity
- avoid parallel motion that hides the main event
- no critical meaning should exist only in animation

## Output

- state transitions
- motion intent
- reduced-motion fallback
- ambiguity risks
