# Accessibility And Visual Systems / 无障碍与视觉系统

## 1. Accessibility Is Baseline Quality

Always account for:

- contrast
- focus visibility
- keyboard navigation
- target size
- reduced motion

## 2. Visual System Basics

- consistent spacing rhythm
- typography hierarchy
- controlled color roles
- explicit component states

## 3. Review Questions

- where would focus get lost
- where is contrast weakest
- what visual token is inconsistent
