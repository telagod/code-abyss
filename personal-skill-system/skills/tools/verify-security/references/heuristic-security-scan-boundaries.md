# Heuristic Security Scan Boundaries / 启发式安全扫描边界

## 1. This Is A Triage Tool

It is good for:

- obvious dangerous patterns
- secret-like material
- suspicious sinks

It is not a replacement for:

- full source-to-sink analysis
- authz reasoning
- exploit validation

## 2. Review Questions

- what does this tool see directly
- what important security property does it not see at all
