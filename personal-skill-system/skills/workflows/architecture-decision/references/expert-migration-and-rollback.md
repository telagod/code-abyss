# Expert Migration And Rollback

Use this reference when the choice creates a bridge period, coexistence cost, or rollback risk.

## Migration frame

State:

- source state
- target state
- bridge period
- compatibility rule
- cutover trigger
- rollback trigger
- decommission rule

## Rules

- never approve a major decision with no rollback trigger
- compatibility work is real work, not a footnote
- migration success should have explicit evidence, not only hope
